# Templates de Bio para Instagram SEO

10 formulas de bio otimizadas para indexacao no Google e descoberta no Instagram.

## Estrutura dos Campos Indexaveis

```
NAME FIELD (64 caracteres):
[Nome/Marca] | [Keyword Principal] | [Local/Especialidade]

USERNAME (30 caracteres):
@[keyword-se-possivel]

BIO (150 caracteres):
Linha 1: Proposta de valor + keyword
Linha 2: Prova social / Diferencial
Linha 3: CTA claro

CATEGORIA:
Selecionar a mais especifica do nicho
```

---

## Formula 1: Resultado

**Quando usar:** Servicos, consultorias, coaches

**Estrutura:**
```
NAME: [Nome] | [O que faz]
BIO:
[Verbo] + [publico] a [resultado numerico em timeframe]
[Prova social com numero]
[CTA com beneficio]
```

**Exemplo:**
```
NAME: Ana Costa | Leiloes de Imoveis
@anacostaleiloes

Ajudo investidores a encontrar imoveis com ate 50% de desconto
+200 clientes assessorados | R$15M em economia gerada
Consultoria gratuita no link
```

## Formula 2: Local

**Quando usar:** Negocios fisicos, servicos locais

**Estrutura:**
```
NAME: [Marca] | [Servico] [Cidade]
BIO:
[O que faz] + [beneficio] em [cidade/regiao]
[Prova social local]
[CTA com facilitador]
```

**Exemplo:**
```
NAME: Cafe Artesanal | Cafeteria SP
@cafeartesanalsp

Cafe especial torrado na hora em Pinheiros
Nota 4.9 no Google | +500 avaliacoes
Conheca nosso espaco
```

## Formula 3: Autoridade

**Quando usar:** Especialistas, profissionais, palestrantes

**Estrutura:**
```
NAME: [Nome] | [Titulo/Especialidade]
BIO:
[Titulo] | [Area de especialidade]
[Resultado] para [numero]+ [publico]
[Recurso gratis + CTA]
```

**Exemplo:**
```

@drpaulonutri

Nutricionista Esportivo | CRN 12345
Ajudei +3.000 atletas a melhorar performance
Cardapio gratis de 7 dias no link
```

## Formula 4: Criador de Conteudo

**Quando usar:** Influencers, educadores, criadores

## Formula 7: Comunidade

**Quando usar:** Movimentos, grupos, nichos especificos

**Estrutura:**
```
NAME: [Nome do Movimento] | [Nicho]

[CTA para fazer parte]
```

```
NAME: Maes Sem Culpa | Maternidade Real
@maessemculpa

+25.000 maes ja simplificaram a maternidade
Menos culpa, mais conexao com seus filhos
```

## Formula 8: Contrarian

**Quando usar:** Posicionamento diferenciado, disrupcao

**Estrutura:**
```
NAME: [Nome] | [Nicho]
BIO:
[Afirmacao contraria ao senso comum]

[Prova + CTA]
```

**Exemplo:**
```

@joaomarketing

Voce NAO precisa de 10k seguidores pra vender
Ensino a converter com menos de 1.000
+R$500k gerados por alunos | Metodo no link
```

## Formula 9: PAS Compacto

**Quando usar:** Solucao de problemas especificos

**Estrutura:**
```
NAME: [Nome/Marca] | [Solucao]
BIO:
Cansado de [problema comum]?
[Agitacao: consequencia do problema]
[Solucao + CTA]
```

**Exemplo:**
```
NAME: Sono Bom | Especialista em Sono
@sonobom

Cansado de acordar mais cansado que dormiu?
Insonia rouba sua energia e produtividade
Protocolo de 21 dias no link
```

## Formula 10: Nicho Especifico

**Quando usar:** Servicos B2B, nichos muito especificos

**Estrutura:**
```
NAME: [Nome] | [Nicho Especifico]
BIO:
[Profissao]: [promessa especifica]
[Metodo ou framework proprietario]
[Numero de resultados] | [CTA]
```

**Exemplo:**
```
NAME: Dra. Lucia | Advogada p/ Medicos
@draluciadvmed

Advogados: defesa em processos eticos
Metodo Blindagem Medica em 3 passos
+180 medicos protegidos | Consultoria no link
```

## Instrucoes de Personalizacao

1. **Escolha a formula** mais adequada ao seu modelo de negocio
2. **Adapte os textos** mantendo a estrutura
3. **Respeite os limites**: 64 chars (name), 150 chars (bio)
4. **Teste com busca**: pesquise sua keyword no Instagram e veja se aparece
5. **Atualize trimestralmente**: revise numeros e ofertas
